Sonde Nepal API Documentation

REST API for radiosonde telemetry, atmospheric data, and flight predictions.

Base URL: https://sonde.kushal-kc.com.np Version: 1.0

Endpoints

  1. POST /api/ingest/ — Ingest telemetry
  2. GET /api/sondes/ — Latest sonde
  3. GET /api/pred/ — Flight prediction
  4. GET /data/data_by_date/ — Sonde list by date
  5. GET /data/sonde_data/ — Sonde telemetry
  6. GET /data/sonde_KML/ — Download KML
  7. GET /data/skewt/ — Skew-T diagram
  8. GET /data/hodograph/ — Hodograph
  9. GET /data/emagram/ — Emagram
  10. GET /data/stuve/ — Stuve diagram
  11. GET /data/tephigram/ — Tephigram
  12. GET /data/atmosphere/ — Atmospheric indices
  13. GET /history/data/ — Recent sondes
  14. GET /predict_prox/ — Tawhiri proxy

Authentication

Protected endpoints require an API key sent via the X-API-KEY header. Get your key from the Dashboard after signing up.

X-API-KEY: YOUR_API_KEY

Endpoints marked No Auth work without a key.

1. Ingest Telemetry

POST /api/ingest/ Auth Required

Submit a single telemetry frame. Creates or updates a sonde record and triggers an automatic flight prediction.

Request Body (JSON)

FieldTypeRequiredDescription
idstringYesSonde identifier
datetimestringYesISO 8601 timestamp
latfloatNoLatitude
lonfloatNoLongitude
altfloatNoAltitude (m)
tempfloatNoTemperature (°C)
humidityfloatNoHumidity (%)
pressurefloatNoPressure (hPa)
vel_hfloatNoHorizontal velocity (m/s)
vel_vfloatNoVertical velocity (m/s)
headingfloatNoHeading (degrees)
battfloatNoBattery voltage
f_centrefloatNoCentre frequency
snrfloatNoSignal-to-noise ratio
typestringNoSonde type (e.g. RS41)
subtypestringNoSonde subtype
freqstringNoFrequency
frameintNoFrame number

Python

import requests

url = "https://sonde.kushal-kc.com.np/api/ingest/"
headers = {"X-API-KEY": "YOUR_KEY", "Content-Type": "application/json"}

payload = {
    "id": "IMET5-55095407",
    "datetime": "2026-05-04T00:15:00Z",
    "lat": 27.682,
    "lon": 85.288,
    "alt": 15000,
    "temp": -45.2,
    "humidity": 12.0,
    "pressure": 100.0,
    "vel_h": 15.3,
    "vel_v": 5.1,
    "heading": 225.0,
    "batt": 3.8,
    "f_centre": 403000000,
    "snr": 18.5
}

r = requests.post(url, json=payload, headers=headers)
print(r.json())  # {"ok": true}

Response

{"ok": true}

2. Get Latest Sonde

GET /api/sondes/ No Auth

Returns all telemetry frames for the most recently received sonde, ordered by frame number.

Python

import requests

r = requests.get("https://sonde.kushal-kc.com.np/api/sondes/")

print(f"ID: {sonde['sonde_id']}")
print(f"Frames: {len(sonde['data'])}")
print(f"Latest: {sonde['data'][0]['lat']}, {sonde['data'][0]['lon']}")

3. Get Flight Prediction

GET /api/pred/?sonde_id=ID No Auth

Returns the Tawhiri flight prediction (ascent + descent trajectory) for a given sonde.

ParameterTypeRequiredDescription
sonde_idstringYesSonde identifier

Python

import requests

r = requests.get("https://sonde.kushal-kc.com.np/api/pred/",
                  params={"sonde_id": "IMET5-55095407"})
pred = r.json()

# Ascent trajectory
ascent = pred["data"]["prediction"][0]["trajectory"]
# Descent trajectory
descent = pred["data"]["prediction"][1]["trajectory"]

landing = descent[-1]
print(f"Landing: {landing['latitude']}, {landing['longitude']} @ {landing['altitude']}m")

curl

curl.exe "https://sonde.kushal-kc.com.np/api/pred/?sonde_id=IMET5-55095407"

4. Get Sonde List by Date

GET /data/data_by_date/?date=YYYY-MM-DD Auth Required

Returns all sondes launched on a given date.

ParameterTypeRequiredDescription
datestringYesDate in YYYY-MM-DD format

Python

import requests

url = "https://sonde.kushal-kc.com.np/data/data_by_date/"
headers = {"X-API-KEY": "YOUR_KEY"}
params = {"date": "2026-05-04"}

r = requests.get(url, headers=headers, params=params)
for s in r.json()["data"]:
    print(f"{s['sonde_id']} — {s['date']}")

6. Download KML Path

GET /data/sonde_KML/?sonde_id=ID Auth Required

Downloads a KML file with the sonde's 3D flight path. Open in Google Earth or import into GIS software.

ParameterTypeRequiredDescription
sonde_idstringYesSonde identifier

Python (download file)

import requests

url = "https://sonde.kushal-kc.com.np/data/sonde_KML/"
headers = {"X-API-KEY": "YOUR_KEY"}
params = {"sonde_id": "IMET5-55095407"}

r = requests.get(url, headers=headers, params=params)

with open("sonde_path.kml", "wb") as f:
    f.write(r.content)
print("Downloaded sonde_path.kml")

curl

curl.exe "https://sonde.kushal-kc.com.np/data/sonde_KML/?sonde_id=IMET5-55095407" -H "X-API-KEY: YOUR_KEY" -o sonde.kml

13. Recent sondes

GET /history/data/ No Auth

Returns the 5 most recent sondes with full telemetry. Used by the History page.

Python

import requests

r = requests.get("https://sonde.kushal-kc.com.np/history/data/")
history = r.json()

for sonde in history["data"]:
    print(f"{sonde['sonde_id']} — {sonde['date']} — {len(sonde['data'])} frames")

14. Flight Prediction Proxy

GET /predict_prox/ No Auth

Proxies requests to the SondeHub Tawhiri prediction API. Pass any Tawhiri parameters as query strings.

ParameterTypeRequiredDescription
launch_datetimestringNoISO 8601 launch time
launch_latitudefloatNoLaunch latitude
launch_longitudefloatNoLaunch longitude
launch_altitudefloatNoLaunch altitude (m)
ascent_ratefloatNoAscent rate (m/s)
burst_altitudefloatNoBurst altitude (m)
descent_ratefloatNoDescent rate (m/s)

Python

import requests

r = requests.get("https://sonde.kushal-kc.com.np/predict_prox/", params={
    "launch_datetime": "2026-05-04T00:00:00Z",
    "launch_latitude": 27.682,
    "launch_longitude": 85.288,
    "launch_altitude": 1350,
    "ascent_rate": 5.0,
    "burst_altitude": 30000,
    "descent_rate": 5.0
})
prediction = r.json()
print(prediction)

Rate Limits & Notes

  • No rate limits are currently enforced, but be respectful.
  • The /api/sondes/ endpoint returns only the latest sonde. Use /data/data_by_date/ + /data/sonde_data/ for historical data.
  • Image endpoints (/data/skewt/, etc.) return PNG files directly.
  • Flight predictions are generated automatically when telemetry is ingested via /api/ingest/.
  • Get an API key at Dashboard (sign up required).
